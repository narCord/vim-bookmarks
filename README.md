# vim-bookmarks
Plugin de neovim para gestionar bookmarks permanentes en un buffer.  
<br><br>

## Funcionamiento
\<lider\> + ba: Crea un bookmark.

\<lider\> + bl: Muestra la ventana flotante con los bookmarks de buffer.

\<lider\> + bd: Borra el bookmark de la linea donde está el cursor.
